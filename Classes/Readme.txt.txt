Excel sheets for the individual classes will appear hear once the admin assistant has entered the meal choices.
(1) A teacher/class will enter all the meal choices. 
(2) All pupils need to be complete. 
(3) Once all pupils are complete then the send and close button will become available. This will return the meal
counts to admin for use.

NOTE:
When (3) is complete the spreadsheet will close and become unavailable!
